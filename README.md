# StayShield AI

Deep Learning-based Hotel Booking Cancellation Prediction System

StayShield AI is an end-to-end machine learning application that predicts the likelihood of hotel booking cancellations using an Artificial Neural Network (ANN) built with TensorFlow and deployed through Streamlit.

## Features

- Real-time cancellation prediction
- Booking stability assessment
- Risk categorization
- Probability scoring
- Smart recommendations
- Interactive Streamlit UI

## Tech Stack

- Python
- TensorFlow / Keras
- Scikit-Learn
- Pandas
- NumPy
- Streamlit
